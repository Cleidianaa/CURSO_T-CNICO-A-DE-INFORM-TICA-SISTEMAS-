Visual C#
Windows Forms

Controlos:

TextBox - É um controlo que permite receber os dados do utilizador.

Todos os controlos possuem propriedades.

	Name - Identificar o controlo(nome do controlo)
	Text - É a propriedade que guarda a informação do controlo.
	Font - Formatar o aspeto do tipo de letra, o estilo e o tamanho.
	Forecolor - A cor do tipo de letra.
	Backcolor - A cor de fundo do controlo.
	
	Clear() é o método para limpar o Text da TextBox.
	
Form:
	Name
	Text	
	FormBorderStyle - Sizable(permite redimensionar), FixedDialog(não redimensiona)
	MaximizeBox - True (False para não maximizar)
	
Button:
	Name, Text, Font, Backcolor, Forecolor 	
	
Label: É usado para display.
	Name, Text, Font, Backcolor, Forecolor 
	BorderStyle - Permite definir um limite para a Label.
	Autosize - Redimensionamento automático ou manual.
	
	ResetText() é o método para limpar o Text da Label.
	

Eventos	ou acontecimentos. São ações que irão ser acionados ou por um clique,
por duplo clique, ao pressionar o Enter.
